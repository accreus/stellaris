# Test Platform

This is a full-stack monorepo managed with [pnpm workspaces](https://pnpm.io/workspaces).

## Project Structure

- `apps/backend/` - The backend application
- `apps/frontend/` - The frontend application
- `packages/shared/` - Shared code, types, and utilities used across apps

## Getting Started

### Prerequisites

- Node.js
- [pnpm](https://pnpm.io/installation)

### Installation

Install dependencies from the root of the workspace:

```bash
pnpm install
```
